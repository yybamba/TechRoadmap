How to Use This School
Read
Start with the Foundational Essay above. Then pick a pillar that matches where you are building right now. Read the anchor pieces for that pillar.

Open a pillar page to see all topics in that domain. Follow the serialized series in order — each part builds on the previous.

Join the conversation on LinkedIn or Substack. The school is a two-way thing — the best readers sharpen the ideas that come back as new essays.
Build in Public
The school documents what has been built — the multi-tenant payment infrastructure, built in public. Build on top of these ideas. Share what you make.
